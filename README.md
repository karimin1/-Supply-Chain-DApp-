# 📦 Supply Chain dApp (Blockchain Tracking System)

## 📌 Overview
This project is a decentralized Supply Chain application built using Solidity and Web3 technologies.  
It allows tracking of products through different stages (creation, shipping, delivery) on the blockchain, ensuring transparency and trust.

---

## ⚙️ Features
- 📦 Create products on-chain
- 🚚 Track product lifecycle (Created → Shipped → Delivered)
- 👤 Role-based actions (Owner / Supplier / Buyer)
- 🔍 Transparent product history
- 🔐 Secure and tamper-proof records

---

## 🛠️ Tech Stack
- Solidity (Smart Contracts)
- Hardhat
- Ethers.js
- React / Next.js (Frontend)
- Ethereum (Sepolia Testnet)

---

## 🚀 How It Works
1. Manufacturer creates a product on the blockchain  
2. Product status is updated at each stage  
3. All transactions are recorded on-chain  
4. Anyone can verify product history
